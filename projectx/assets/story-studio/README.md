# Story Studio Media Assets

Place generated media files here and update `projectx/data/story-studio/stories.json`.

Recommended paths:

- `audio/project-x-story.mp3`
- `video/project-x-story.mp4`
- `infographics/project-x-architecture.png`
- `transcripts/project-x-story-transcript.md`

Story Studio Sprint 1.1 includes a media loader. If a file is listed in `stories.json`, it will appear automatically in the Read / Listen / Watch / Explore workspace.
