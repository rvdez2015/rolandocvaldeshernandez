# Privacy and Data Governance

## Core Principle
Project X is designed using Privacy by Design.

Student identity is not part of the Curriculum Repository.

## Data Classification

| Classification | Examples | Stored in Curriculum Repository |
|---|---|---|
| Public | OCR specifications, National Curriculum | Yes |
| Professional | Schemes of Work, lesson plans, resources, CPD certificates | Yes |
| Operational | Reflections, lesson analytics, AI context | Yes |
| Aggregated | Class-level averages, common misconceptions | Yes |
| Personal | Student names, emails, dates of birth | No |
| Sensitive | SEND, medical, safeguarding, behaviour case notes | No |

## Professional Standards Centre Rule
The Professional Standards Centre may store professional certificates and training evidence. It must not store pupil-identifiable data, safeguarding case notes, SEND information, medical details or behaviour records.
