# Project X v4.0 — Release 1.0 Alpha Addition

## Professional Standards Centre — Safeguarding Portfolio

This release addition creates a safe professional compliance space for safeguarding certificates.

## Included

- New `apps/professional-standards/` module.
- Safeguarding certificate metadata repository.
- Searchable and filterable certificate dashboard.
- Certificate document store under `professional/safeguarding/`.
- Sidebar navigation entry.
- ADR-0010 — Professional Standards Centre.
- Privacy and Data Governance document.

## Imported Certificates

14 certificate files were imported into the safeguarding portfolio.

## Privacy Position

This module stores professional evidence only. It must not store pupil names, safeguarding case notes, SEND records, medical information or behaviour-sensitive records.
